# ZiShare Landing Page

Simple landing page with waitlist form for market validation.

## Features

- Modern, responsive design
- Waitlist form (stores data in localStorage - will be connected to backend)
- Key value propositions highlighted
- Market statistics display

## Usage

Open `index.html` in a browser or serve with any static file server.

## Next Steps

Once backend is ready, connect the form to an API endpoint to store waitlist entries in the database.
