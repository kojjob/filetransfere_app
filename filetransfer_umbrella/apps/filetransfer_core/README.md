# ZipShare Core

Business logic and data layer for the ZipShare file transfer platform.

## Domains

- **Accounts** - User authentication and management
- **Transfers** - File transfer records and state
- **Chunks** - File chunking for large uploads
- **Sharing** - Share links and access control
- **Waitlist** - Pre-launch waitlist management
- **Usage** - Usage statistics and analytics
- **Api** - API key management
